# dotfiles

## prerequirement

install zsh + tmux + vim

install oh-my-zsh

vim install plugins by Plug


```bash
# install by git
git clone https://github.com/realcaiying/dotfiles.git .dotfiles && cd .dotfiles && source install.sh

# install without git
cd; curl -#L https://github.com/realcaiying/dotfiles/tarball/master | tar -xzv --strip-components 1 --exclude={README.md,install.sh}
```
